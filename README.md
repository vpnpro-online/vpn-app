# VPN Pro

Fast and private VPN client for Windows, macOS, Linux, Android and iOS.

## Downloads

Get the latest build from the [Releases](https://github.com/vpnpro-online/vpn-app/releases/latest) page.

## Platforms

- **Android**: `.apk` — install directly (enable "Install unknown apps" for your browser)
- **Windows**: `.zip` — extract and run `Hiddify.exe`
- **Linux**: `.AppImage` / `.deb` / `.rpm`
- **macOS**: `.dmg` (coming soon)
